## setup
Environment for this project uses Python >= 3.8.
```
virtualenv env -p /path/to/bin/python3.8
pip install -i requirements.txt
```
For developers:
```
virtualenv env -p /path/to/bin/python3.8
pip install -i dev-requirements.txt
```
